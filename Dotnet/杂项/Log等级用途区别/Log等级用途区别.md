| 日志方法	枚举值	使用场景 / 记录内容                    |
| ------------------------------------------------------------ |
| Log.Verbose()	0	最详细的调试信息，通常只在开发或排查问题时使用。 |
| 例如：方法进入/退出、循环内部状态、临时变量值等。            |
| Log.Debug()	1	调试级别信息，用于跟踪程序逻辑和流程。   |
| 例如：关键变量值、状态变化、函数调用参数等。                 |
| Log.Information()	2	普通运行信息，用于记录程序运行情况。 |
| 例如：用户登录、操作完成、服务启动/停止等。                  |
| Log.Warning()	3	警告信息，不影响程序运行，但可能是潜在问题。 |
| 例如：配置缺失、外部服务响应慢、非关键异常被捕获等。         |
| Log.Error()	4	错误信息，程序出现问题，需要修复。       |
| 例如：数据库操作失败、文件读写异常、网络请求失败等。         |
| Log.Fatal()	5	严重错误，导致程序无法继续运行。         |
| 例如：系统启动失败、关键服务异常崩溃、不可恢复的异常等。     |
