这种方法直接修改 `logind.conf`，无论你是否登录桌面，设置都会生效。

1. **打开配置文件**：
   **Bash**

   ```
   sudo nano /etc/systemd/logind.conf
   ```
2. **找到并修改以下参数**： 在文件中找到这些行（如果前面有 `#`，请删掉它），并将值改为 `ignore`：

   * `HandleLidSwitch=ignore`（合盖时的操作）
   * `HandleLidSwitchExternalPower=ignore`（插着电源合盖时的操作）
   * `HandleLidSwitchDocked=ignore`（外接显示器/底座时的操作）

   **修改后的样子：**
   **Plaintext**

   ```
   [Login]
   HandleLidSwitch=ignore
   HandleLidSwitchExternalPower=ignore
   HandleLidSwitchDocked=ignore
   ```
3. **保存并退出**： 按下 `Ctrl + O` 保存，`Enter` 确认，然后 `Ctrl + X` 退出。
4. **应用更改**： 运行以下命令重启服务（这可能会导致你短暂退出登录）：
   **Bash**

   ```
   sudo systemctl restart systemd-logind
   ```
